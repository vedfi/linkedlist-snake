LinkedList Snake Game v1.0

Builded with C & SDL2 Libraries
Works Only With 64-Bit Machines

-UP DOWN LEFT AND RIGHT TO MOVE
-ESC TO EXIT

--- FOR LINUX ---
-Install libSDL2 & libSDL2_mixer to your system using a package manager like apt.
-Set linker settings on your C IDE.
-Compile & RUN.
